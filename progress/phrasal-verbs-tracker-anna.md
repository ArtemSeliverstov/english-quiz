## Russian-calque traps for Anna

Anna's documented L1 weakness is **preposition swaps and dropped prepositions** (per `family-profiles.md` and the 2026-04-30 stats-review). PV-related calques to watch:

| Russian | Calque to avoid | Correct PV |
|---|---|---|
| искать | search → bare *look* + object | **look for** |
| ждать | bare *wait* + object | **wait for** |
| слушать | bare *listen* + object | **listen to** |
| зависеть | depend **from** / **of** | depend **on** |
| смотреть за / присматривать | look **for** the kids (search!) | **look after** the kids (care) |
| убрать | put behind / put on side | **tidy up** / **put away** / **clean up** |
| подождать секунду | wait second | **hold on** a second |
| забрать (детей из школы) | take from school | **pick up** (the kids from school) |
