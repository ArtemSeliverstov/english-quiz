## Session 2 — Смешанные числа ↔ неправильные дроби (2026-07-15, CC chat)

Первый урок трека, **player-initiated** («это Ника, давай первый урок»). Тема — кандидат #1 из S1. Обучающий формат (не тест): концепция через пиццы → арифметический способ → разбор на месте.

### Прошли

| Шаг | Итог |
|---|---|
| Разминка: сократи 12/8 | 3/2 сходу ✅ |
| Неправильная → смешанная (пиццы + «раздели верх на низ») | 5/2 = 2½ ✅, 7/3 = 2⅓ ✅ (другой знаменатель) |
| Смешанная → неправильная («целую × низ + верх») | 2¾ = 11/4 ✅ |
| Капстоун 10/4 → смешанным | 2 2/4 → после напоминания 2½ |

### Ошибки — две, обе показательные

1. **«3/2 = 1 целая и 4»** в самом начале — потерялась в остатке (должна была быть 1 половинка). Медленный пересчёт по пиццам снял, дальше чисто.
2. **Капстоун: 2 2/4 вместо 2½** — конверсию сделала верно, но не сократила дробную часть. **Ровно хвост S1** («не доводит до конца»). Разобрали как «фирменный шаг».

### Вывод

Смешанные числа ↔ неправильные — оба направления, включая незнакомый знаменатель, за один урок на уровне «понял». Обучаемость снова высокая. Хвост «сократи в конце» устойчив across S1→S2 — вплетать «проверь дробную часть» в каждый ответ до автоматизма. Тема требует интервального повтора (вернуться через ~1–2 недели холодной задачей). Следующая по кандидатам: сравнение дробей / числовая прямая, либо мост дроби↔проценты.

## Session 3 — Отношения, метод частей (2026-07-22, CC chat)

Запрос от Артёма: Ника не понимала задачу №4 из демоварианта Технолицея (`Nicole/6-7 dol.pdf`) — отрезок MN, MA : AB : BN = 2 : 3 : 7, BN = 4,9 см, найти MN. Формат: объяснение записи a : b : c через «одинаковые кубики» → 8 вопросов лесенкой с проверкой понимания с разных сторон, по одному за раз.

### Лесенка

| Шаг | Проверялось | Итог |
|---|---|---|
| 4 : 1 — кто длиннее и во сколько раз | чтение отношения | ✅ |
| 2 : 3 : 5 — сколько частей всего | сумма частей | ✅ |
| 6 см и 9 см → записать отношение | обратный ход (2 : 3) | ✅ |
| 3 : 4, первый кусок 15 см → одна часть | размер части | ❌ → ✅ (см. ошибку) |
| 2 : 5, большой 20 см → вся верёвка | полная цепочка, 2 куска | ✅ 28 |
| 1 : 4 : 5, средняя 12 см → весь отрезок | полная цепочка, 3 куска | ✅ 30 |
| MA : AB : BN = 2 : 3 : 4, BN = 12 → MN | формат контрольной, целые | ✅ 27 |
| **Оригинал: 2 : 3 : 7, BN = 4,9 → MN** | то же + десятичные | ✅ **8,4 сама** |

### Ошибка — одна

Шаг «одна часть»: ответила 20 вместо 5 — посчитала **второй кусок** (4 × 5), перескочив через вопрос. Не концептуальная: после повторного указания «три кубика = 15 см» мгновенно дала 5. Диагноз — читает вопрос по диагонали, знакомый паттерн «спешит вперёд».

### Вывод

Метод частей (найди кубик → посчитай кубики → умножь) взят за одну сессию: от «не понимаю задачу» до самостоятельного решения оригинала с десятичными. Ошибок в вычислениях ноль, включая 4,9 : 7. Обучаемость стабильно высокая, паттерн S1–S3 общий: концепции схватывает с одного объяснения, риски — невнимательное чтение вопроса и недоведение до конца. Тема — часть A2 (пропорции); для закрепления дать холодную задачу через ~неделю (вариант: известна не крайняя, а средняя часть, или известна разность кусков).
