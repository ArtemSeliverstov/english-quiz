The Holistic Plan (4 Phases) PHASE 1 Fix Production on Existing PVs + Rebuild Foundation Weeks 1–3 Goal: Raise input accuracy from 50% → 70%. Fix the 26 chronic/struggling PVs. Strengthen A2 foundation through exercise exposure. QUIZ Changes Add gap "scaffolds" for 7 hardest input PVs: For bring about, come up with, read up on, drill down into, get out of, cut down on, take up — create parallel gap versions with close distractors (the wrong particle choices you actually make). These build the particle-meaning association before facing cold production. When gap accuracy reaches 2× correct, the input version re-enters rotation via weak-questions mode. Add A2 PVs as distractors in existing B1/B2 gap questions: Where possible, add A2-level wrong options to existing gap questions. E.g., for a get through question, add get in and get back as distractors — this forces you to distinguish them. Slot 9 (Free Write email): Give 3 target PVs from the chronic list + 2 from A2 level. Business email scenario. Must use all 5.
